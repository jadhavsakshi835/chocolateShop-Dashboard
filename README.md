# 🍫 Choco Bliss Sales Dashboard (Power BI)

## 📌 Project Overview
The Choco Bliss Sales Dashboard is an interactive Power BI dashboard created to analyze chocolate sales data. It provides insights into sales performance, product demand, revenue distribution, and salesperson performance across different countries.

## 🎯 Objectives
- Analyze overall sales performance
- Monitor monthly sales trends
- Compare orders across countries
- Identify best-selling and least-selling products
- Analyze revenue by sales channel
- Evaluate salesperson performance

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

## 📊 Dashboard Features
- Total Sales KPI
- Average Discount
- Marketing Spend
- Sales Trend Over Time
- Country-wise Orders
- Best Selling Products
- Least Selling Products
- Revenue by Sales Channel
- Best Performers
- Country Filter (Slicer)

## 📈 Key Insights
- Track monthly sales performance.
- Compare sales across different countries.
- Identify the highest and lowest-selling products.
- Understand revenue contribution from different sales channels.
- Evaluate top-performing salespersons.
